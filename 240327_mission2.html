웹 페이지 기본 구조와 작성 방법 미니테스트

01 다음 괄호 안을 채우시오.

(요소/element/component)는 HTML 페이지를 구성하는 각 부품입니다.

(태그)는 요소를 만들 때 사용하는 작성 방법입니다.

(주석)은 프로그램 실행에는 영향을 미치지 않고 설명용으로 사용하는 코드를 의미합니다.

(스타일시트(Cascading style sheets))는 마크업 언어가 표시되는 방법을 기술하는 언어로 HTML 페이지의 스타일을 지정할 때 사용합니다.

(자바스크립트(Javascript))는 HTML 페이지에서 사용하는 프로그래밍 언어를 의미합니다.


02 다음 질문에 OX를 표시하시오.

(1) HTML 태그는 반드시 내부에 무언가를 포함해야 합니다. (X)

br, hr

(2) HTML 태그 내부에는 텍스트만 입력할 수 있습니다. (X)

(3) html 태그의 lang 속성은 구글 등 검색 엔진에 문서의 언어 정보를 알려 줄 때 사용됩니다. (O)


03 다음 HTML 요소의 속성 이름과 속성 값, 태그 이름을 모두 적으시오.

<img title="바다" alt="sea" src="sea.png">
  • 태그 : img
  • 속성 : title, alt, src
  • 값 : "바다", "sea", "sea.png"

04 다음 중 head 태그 내부에 입력할 수 없는 태그는?

① style 태그 ② meta 태그 ③ base 태그 ④ header 태그


05 HTML5 기본 페이지 형태(구조)를 하단 공란에 적으시오.

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Document</title>
</head>
<body>
</body>
</html>

06 다음 코드에서 잘못된 부분을 찾아 바르게 수정하시오.

<!DOCTYPE html>
<html>

<head>
<title>HTML5 Basic</title>
<style src="outer_stylesheet.css"></style>
<script src="outer_javascript.js"></script>
</head>

<body>
</body>

</html>

수정되어야 할 구문 : 

link rel="stylesheet" href="./style.css"
